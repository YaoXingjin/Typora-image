# Typora-image

这个仓库用作 Typora 的图床，用于存储和管理 Typora 中使用的图片。

## 🤔 已经有 PicGo 了，还需要用自定义脚本吗？

**简短回答：看你的需求和技术水平。**

- **新手用户，追求简单**：推荐使用 PicGo，图形界面更友好
- **技术用户，追求控制**：推荐使用自定义脚本，完全可控
- **团队协作**：PicGo 更标准化
- **个人使用且重视隐私**：自定义脚本更合适
